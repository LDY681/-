严教：出牌阶段限一次，你可以选择一名其他角色。  
从牌堆顶亮出四张牌，该角色将这些牌分成点数之和相等的两组，你与其各获得其中一组，然后将剩余未分组的牌置入弃牌堆。  
若未分组的牌超过一张，你本回合手牌上限-1。  

省身：当你受到伤害后，你可以摸一张牌且下一次发动“严教”亮出的牌数+1。  
若你的手牌数为全场最少，则改为摸两张牌；若你的体力值为全场最少，则“严教”亮出的牌数改为+2（加值总数不能超过4）。
